
# Return type of variables in JavaScript

1) Primitive Datatypes => Stack Memory
       Number => number
       String  => string
       Boolean  => boolean
       null  => object
       undefined  =>  undefined
       Symbol  =>  symbol
       BigInt  =>  bigint

2) Non-primitive Datatypes => Heap Memory
       Arrays  =>  object
       Function  =>  function
    Object  =>  object
